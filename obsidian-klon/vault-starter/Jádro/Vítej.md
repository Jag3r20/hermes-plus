# Vítej v SOCIYA neural vault 👋

Tohle je tvůj nový **druhý mozek**. Je to úplně prázdný startovní vault — všechno tady si přizpůsob.

## Filozofie „sluneční soustavy"
- ☀️ **Slunce = složka `Jádro`** — sem patří klíčové info (o tobě/firmě, principy, kontakty).
- 🪐 **Planety = složky** nejvyšší úrovně (projekty, klienti, oblasti...).
- 🌙 **Měsíce = poznámky** uvnitř planety.

Otevři nahoře **soustava / síť** a přepínej **2D / 3D**.

## Jak začít
1. Smaž tyhle ukázkové poznámky.
2. Založ si vlastní složky (planety) a poznámky.
3. Propojuj je `[[odkazy]]` a znač `#tagy`.
4. Napoj Clauda přes MCP (viz README) — bude ti s vaultem pomáhat.
